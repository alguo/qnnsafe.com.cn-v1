qnnsafe.com-v1
==============

Official QNN Safe website in Chinese (Mid 2013).

✔SCSS
✔CoffeeScript
✔Haml
✔Responsive
✔Static

Build
-----

1. Install Ruby 2.0.
2. Install bundler: ``gem install bundler``
3. Install necessary gems: ``bundle``
4. Build site: ``rake build``.

Run ``rake`` to see available tasks.
